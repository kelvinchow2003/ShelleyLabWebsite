Drop your Shelley Automation logo files here:

  shelley-icon.png       — the standalone swoosh (used in the nav + login screen)
  shelley-wordmark.png   — the full "ShelleyAutomation" lockup (optional; used in the nav)

PNG (transparent background preferred) or SVG both work. If you use SVG, update
the file extensions in src/components/Logo.tsx (ICON_SRC / WORDMARK_SRC).

Files placed here are served from the site root, e.g. /shelley-icon.png
Until you add them, the app shows a built-in SVG fallback logo.
